# **App Name**: DriverShield

## Core Features:

- Secure Authentication: User authentication via email/password, with email verification required for account activation.
- Centralized Dashboard: Dashboard provides navigation to search, report submission, account details, search history, and report history.
- Advanced Search: Enables searching for drivers by name or code and displays comprehensive driver information from the database.
- Report Submission: Allows authorized users to submit reports with driver details, categories, tags, comments, and optional image uploads.
- Account Management: Provides access to company registration details, submitted reports, and search history. Also, the feature where users can only delete the info they submitted themselves.
- Payment Integration: Annual payment integration before access is granted.
- User Alert System: Automated alerts, like the welcome message to teach the users about the platform.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and security.
- Background color: Light gray (#ECEFF1), a subtle background to ensure readability and focus on content.
- Accent color: A muted violet (#9575CD) for interactive elements. This choice provides enough contrast without being obtrusive.
- Clean and modern font for headers to provide clarity.
- Readable and clear sans-serif font for the main text to be accessible.
- Simple, professional icons for easy navigation
- Consistent and intuitive layout across all pages.
